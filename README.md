# Arduino Digital & Analog Sensors

##  Project Overview
This project demonstrates the difference between digital and analog sensors using Arduino.

---

##  Digital Sensor (PIR Motion Sensor)
- Detects motion
- Output: HIGH / LOW
- Turns the LED ON when motion is detected

---

##  Analog Sensor (Potentiometer)
- Provides continuous values (0–1023)
- Controls LED brightness using PWM

---

##  Components
- Arduino Uno
- PIR Motion Sensor
- Potentiometer
- LED
- Resistors
- Breadboard

---

##  Key Concepts
- Digital vs Analog signals
- analogRead() vs digitalRead()
- PWM using analogWrite()

---

##  Simulation
Designed and tested using Tinkercad
